![License](https://img.shields.io/badge/License-MIT-lightgrey)

Backend сервис для системы управления сетью симрейсинг-клубов **AndefRacing**.

Приложение предоставляет REST API для клиентов, сотрудников, администраторов и управляющих клубов.  
Система позволяет управлять клубами, бронированиями симуляторов, персоналом и получать статистические отчеты.

---

# Технологии

- **Java**
- **Spring Boot**
- **Spring Security**
- **JWT Authentication**
- **PostgreSQL**
- **Redis**
- **Hibernate (JPA)**
- **MapStruct**
- **Maven**
- **Docker / Docker Compose**
